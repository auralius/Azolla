Dependencies:

Boost Library http://www.boostpro.com/download/
FLTK Library  http://www.fltk.org/software.php?VERSION=1.0.11&FILE=fltk/1.0.11/fltk-1.0.11-source.tar.bz2

For FLTK library, if you don't want to compile yourself, you can find in my sites:
https://sites.google.com/site/auraliusproject/new-azolla

Add these items in your environment variables:

BOOSTROOT
FLTKROOT

Set them to point to Boost and FLTK root directory respectively.
